## Read: on-spec quality dose, but ran a touch hot in the heat

- **Structure confirmed:** the characterizer's interval_guess (4 work reps, ~240 s work / ~243 s recovery) matches the prescribed 4×4 exactly — a clean sawtooth, not a steady run with incidental surges. Zones: easy ~46 % (warm-up + jog recoveries), threshold ~30 %, VO2 ~9 %, steady ~10 %, sub-threshold ~6 %.
- **It was run as VO2 — by group design, confirmed by Patrik.** Prescription was threshold pace (158–163) with a >165 abort, because Jun 2 already banked one 4×4 VO2 week. But the group runs 4×4 *as* VO2, and Patrik ran it that way: HR **drifted 140.4 → 156.2** (4.6 bpm/km) and **peaked 172**, the back reps in VO2. Warm/humid conditions, stiff legs, and ~42 km of commute riding the day before (the Jun 8 "off / easy spin" that ran long) added to the cost and the "stumma ben" — but the intensity itself was deliberate VO2, not just heat drift.
- **Patrik's read, accepted:** the 4×4 VO2 sets cost him about the same in recovery as the group's usual 5×6 min threshold sets, so swapping a threshold week for a VO2 week isn't adding net load — it's shifting the intensity bias (top-end vs lactate-clearance). For a 160 km goal, threshold is the marginally more *specific* of the two, but one short VO2 session as the week's only quality has negligible interference with ultra prep. **The "take it at threshold / >165 abort" rule is retired** as an unrealistic constraint on a group-led session; the real guards are unchanged — keep everything else easy and protect the long run.
- **Load:** suffer 44 / TRIMP 71.6 over ~32 min — a concentrated quality dose, right alongside the Jun 2 lunch 4×4 (suffer 46, 33 min). Appropriate for the week's single hard session.
- **HR source caveat:** per project memory the wrist optical monitor under-reads on intervals, so true rep HRs may run a few bpm hotter than 172. Doesn't change the read — if anything it reinforces "ran hot today."

## Gear: Saucony for intervals, and dropping the Megablast Tuesday role

Patrik ran his habitual interval shoe, the **Saucony Endorphin Speed 4**, and consciously rested the Megablast — with the right reasoning: on VO2 intervals the training stimulus is set by *physiological* intensity (HR/VO2), not by the pace the shoe produces, so the Megablast's propulsion buys faster splits, **not more training effect**. Its limited super-foam life and race-day pop are better saved for where pace matters (races, pace-target tempo).

More to the point: the only reason the Megablast was pinned to Tuesdays was a **fit/suitability test for the 160 km race** — and that pivot is already settled (Megablast demoted, Evo SL Silver is the ultra shoe). So the Tuesday-Megablast role is **unpinned**: Endorphin Speed is the default interval shoe, and Patrik will reach for the Megablast **ad hoc, when he feels like it** rather than on a fixed slot. **Open watch:** he wants to keep tabs on whether the Megablast's character shifts once it's properly broken in (the propulsion/toe-box read so far is on relatively fresh foam) — if it does, the race-shoe and rotation notes get revisited.
